# Future AI Moderator

## Status

Deferred. The initial release uses platform enforcement and Judge conduct findings rather than a separate live Moderator agent.

## Intended responsibility

A future Moderator may supervise debate flow: enforce turn order, manage raise-hand requests, announce transitions, issue conduct warnings, and stop a debate when rules require it.

It must not decide the winner, privately coach a side, or replace API authority. Any future implementation needs a separate specification for deterministic state transitions, human override, audit records, and false-positive handling.
