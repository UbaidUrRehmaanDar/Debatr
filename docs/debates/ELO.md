# Ratings and ELO

## Status

Deferred. Debatr’s initial release is a private learning tool for up to ten users, so ratings are not needed to provide its core value.

## Why defer it

An ELO score creates competitive incentives, requires decisions about draws, inconclusive debates, abandoned debates, rematches, provisional ratings, and anti-gaming protections. Adding it before the debate and judging systems are trustworthy would create misleading results.

## Future requirements

If ratings are introduced, document the exact formula, initial rating, K-factor, eligibility rules, handling of draws/inconclusive/terminated debates, visibility, reset policy, and whether AI-assisted debates should affect a competitive rating at all.

No current schema or API may assume ELO is active.
